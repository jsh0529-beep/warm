    API 키 발급받아 등록하는 법

    상단 검은 바에서 사용할 AI를 고르고, 아래 안내대로 그 AI의 키를 발급받아 붙여넣은 뒤 저장을 누르면 됩니다. 초록 불이 켜지면 준비 완료. AI별로 키가 따로 저장되므로 여러 개 등록해두고 바꿔가며 쓸 수 있습니다.

    ① Claude (Anthropic)

    1. console.anthropic.com 접속 후 가입·로그인 (구글 계정 가능)
    2. 좌측 Billing에서 카드 등록 후 크레딧 충전 (최소 $5, 기사 수십 건 분량)
    3. 좌측 API Keys → Create Key → 생성된 sk-ant- 로 시작하는 키 복사 (한 번만 표시되니 창 닫기 전에 복사)

    ② ChatGPT (OpenAI)

    1. platform.openai.com 접속 후 가입·로그인 (ChatGPT 계정과 동일하지만, 유료 결제는 ChatGPT Plus와 별개로 API 크레딧을 따로 충전해야 합니다)
    2. 우측 상단 톱니바퀴 → Billing에서 카드 등록 후 크레딧 충전 (최소 $5)
    3. API keys 메뉴 → Create new secret key → sk- 로 시작하는 키 복사 (한 번만 표시)

    ③ Gemini (Google)

    1. aistudio.google.com/app/apikey 접속 후 구글 계정으로 로그인
    2. API 키 만들기(Create API key) 클릭 → AIza 로 시작하는 키 복사
    3. 셋 중 유일하게 무료 사용량이 있어 카드 등록 없이 바로 시험해볼 수 있습니다 (분당 요청 수 제한 있음). 단, 기사 이미지 생성은 무료 키로는 안 되고 Google Cloud 결제 계정을 연결해야 열립니다 (장당 약 4~13센트). 본격 사용 시에도 결제 연결을 권합니다

    ④ CMS 자동 입력

    1. 아래 [CMS 자동입력] 버튼을 마우스로 끌어서 브라우저 북마크바에 놓으세요 (최초 1회. 북마크바가 안 보이면 Ctrl+Shift+B)
    2. 기사 생성 후 CMS 입력 복사 버튼 클릭
    3. 언론사 CMS에 로그인해 기사 작성 화면을 연 상태에서, 북마크바의 CMS 자동입력을 클릭 → 제목·부제·본문이 자동으로 채워집니다 (처음 한 번 클립보드 허용 팝업이 뜨면 허용)
    4. 내용 확인 후 저장·출고 버튼은 반드시 직접 누르세요 — 이 도구는 절대 자동 출고하지 않습니다

    📌 CMS 자동입력 ← 이 버튼을 북마크바로 드래그 (업데이트되면 기존 북마크를 지우고 다시 드래그해야 반영됩니다)

    키는 지금 쓰는 이 브라우저에만 저장되며 외부로 전송되지 않습니다. 단, 키를 아는 사람은 누구나 내 크레딧으로 API를 쓸 수 있으니 키를 남에게 공유하거나 GitHub 코드에 직접 넣지 마세요. 유출이 의심되면 각 사이트의 API 키 메뉴에서 해당 키를 삭제하면 즉시 차단됩니다.

    비용은 사용량만큼 차감됩니다. 대략 기사 1건에 Claude·ChatGPT는 10~30센트 수준, Gemini는 무료 한도 내에서 0원부터 가능합니다. 이슈 리스트는 랭킹·RSS 직접 수집이라 어느 AI를 골라도 대부분 무료입니다. 사용량은 각 사이트의 Usage 메뉴에서 확인하세요.
